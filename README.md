# CookWise AI

CookWise AI is a smart kitchen assistant that helps users decide what to cook using the ingredients they already have. Instead of searching through hundreds of recipes, the application suggests meals based on pantry items, ingredient expiry dates, cooking time, nutrition, and personal dietary preferences.

The main goal of the project is to reduce food waste, simplify meal planning, and encourage healthier eating habits.

---

## Features

- User registration and login
- Pantry management
- Ingredient expiry tracking
- AI-based meal recommendations
- Nutrition tracking
- Cooking time-based suggestions
- Personalized dietary preferences
- Smart grocery list generation
- AI recipe generation
- Food waste tracking
- Weekly nutrition insights

## Getting Started

### Clone the repository

```bash
git clone https://github.com/megha-bhagat-04/CookWise-AI.git
```

### Install dependencies

Backend

```bash
cd server
npm install
```

Frontend

```bash
cd client
npm install
```

### Run the project

Backend

```bash
npm start
```

Frontend

```bash
npm run dev
```

## Why CookWise AI?

Planning meals every day can be frustrating, especially when you're unsure what ingredients you have or what needs to be used before it expires. CookWise AI is designed to solve this problem by bringing pantry management, meal recommendations, nutrition tracking, and grocery planning into one place. The aim is to make cooking easier while reducing food waste.
